Library Catalogue

The Library Catalogue contains a database with every book, e-book, journal entry, and article the library has access to. Some courses will require you to use material or excerpts from this database to help your learning or be used as references for your assignments. You can also use this database to find any of these resources you might need to help you during your study. Using the "Read More" button will provide a tutorial on how to use the library catalogue.

Read More

Software

There are computers around campus which contain specific programs for student's use. Engineering and Computing students will require various software which is only available in certain workrooms. Clicking "Read More" will provide you with a map and a list of rooms which contain the necessary software you will need to complete your coursework.
